Wrapper class provides a way to use primitive data types as reference data types.

```Java
public class Main {
	public static void main(String[] args) {
		//primitive      //wrapper
		//---------      //-------
		// boolean       // Boolean
		// char          // Character
		// int           // Integer
		// double        // Double
		
		// Autoboxing: convert primitive to wrapper
		// unboxing: convert wrapper to primitive
		
		// Autoboxing:
		Boolean a = true;
		Character b = '@';
		Integer c = 123;
		Double d = 3.14;
		String e = "Goose";
		
		
		// Unboxing:
	}
}
```
